# ipl-data-analysis
Problem statement:

You are a Senior Data Scientist hired by an IPL franchise’s Analytics Team. The franchise wants data-driven decision-making across match strategy, player selection, venue planning, and fan engagement. You’ve been given the official IPL dataset (matches, seasons, venues, outcomes, basic match-level metadata) covering 2008–2024.

Your task is to analyze the dataset, build predictive and prescriptive models, and deliver clear, actionable recommendations that the team’s Head of Cricket Operations can use before a season. Submit a single Jupyter Notebook that contains the code, visualizations, model outputs, and an executive summary with prioritized business recommendations and assumptions.
